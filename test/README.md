# Taller de ionic

### Instructora: Bertha Brenes
### Duracion: 2horas

## Favor tener instalado:
1. Node
2. Npm
3. Gulp
4. json-server
5. Ionic Cordova

## Al iniciar:
1. tener la base de datos db.json fuera del proyecto de ionic y ejecuatar el comando
```bash
json-server db.json
```
2. Ejecutar la aplicacion de ionic.
```bash
 ionic serve
```
